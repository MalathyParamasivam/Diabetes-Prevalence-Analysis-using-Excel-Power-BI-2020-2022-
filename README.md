
# 🩺 Diabetes Prevalence Analysis Dashboard (2020–2022)

## 📌 Project Overview

This project analyzes global diabetes prevalence data from **2020 to 2022** using **Microsoft Excel** and **Power BI**. The objective is to clean, transform, analyze, and visualize healthcare data to identify trends, regional differences, and key insights through an interactive dashboard.

---

## 🎯 Project Objectives

- Clean and prepare raw healthcare data using Excel.
- Analyze diabetes prevalence by **Region, Country, Age Group, Gender, and Year**.
- Build an interactive Power BI dashboard.
- Identify trends and patterns through descriptive, diagnostic, predictive, and prescriptive analysis.

---

## 📂 Dataset Information

- **Domain:** Healthcare Analytics
- **Time Period:** 2020–2022
- **Data Attributes:**
  - Country
  - Region
  - Year
  - Gender
  - Age Group

## 📈 Key Insights

- Eastern Mediterranean recorded the highest diabetes prevalence.
- Europe recorded the lowest diabetes prevalence.
- The 30+ age group showed higher diabetes prevalence than the 18+ age group.
- Diabetes prevalence showed a gradual increase from 2020 to 2022.
- Identified the Top 10 and Bottom 10 countries based on diabetes prevalence.
